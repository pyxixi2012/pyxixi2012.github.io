配置uwsgi

  • install uwsgi

    $ uwsgi –version
    2.0.13.1

  • uwsgi config file, vi portal-uwsgi.ini

  • run project with uwsgi
    $ uwsgi –ini project-uwsgi.ini

  • tips :
    Address already in use

    sudo rm /tmp/uwsgi.sock

  • start project as service

    create a symbolic link so uWSGI will start our server automatically when the server boots up:

    $ sudo ln -s /home/weihu/web-dev/portal-uwsgi/my-uwsgi.ini /etc/uwsgi/apps-enabled/

    $ sudo service uwsgi start
