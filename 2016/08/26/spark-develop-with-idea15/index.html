Download install file from url :

https://d1opms6zj7jotq.cloudfront.net/idea/ideaIU-15.0.4.tar.gz

tar zvxf ideaIU-15.0.4.tar.gz

mv ideaIU-15.0.4 /opt/ideaIU-15

vi /home/yang/.IntelliJIdea15/idea64.vmoptions

$./idea.sh

NOTE: Config IDEA environment

intall scala ,vim ,sbt ,markdown

File -> Project Structure -> Platform Setting -> Global Libraries

Add scala-sdk-2.10.6 ,spark-assembly-1.6.0-hadoop2.2.0

File -> Project Structure -> Platform Setting -> SDKS

Add JDK ,/usr/java/jdk1.8.0_45

Edit run/debug configurations ,input VM Options : -Dspark.master=local ,# run with local master ,Reference link is : http://blog.csdn.net/shenlanzifa/article/details/42679577

  传递给spark的master url可以有如下几种：

local 本地单线程

local[K] 本地多线程（指定K个内核）

local[*] 本地多线程（指定所有可用内核）

spark://HOST:PORT 连接到指定的 Spark standalone cluster master，需要指定端口。

mesos://HOST:PORT 连接到指定的 Mesos 集群，需要指定端口。

yarn-client客户端模式 连接到 YARN 集群。需要配置 HADOOP_CONF_DIR。

yarn-cluster集群模式 连接到 YARN 集群。需要配置 HADOOP_CONF_DIR。

Run Spark example ,SparkPi (shift + F10)

Q&A

  1. Run with error : Exception in thread “main” java.lang.NoSuchMethodError: scala.collection.immutable.HashSet$.empty()Lscala/collection/immutable/HashSet;

    Scala的版本与Spark版本存在兼容性问题。版本是配套的

    解决措施 :将Scala换成与Spark相匹配的版本。Spark是1.3.0时，该Scala从2.11版本换成2.10版本。
