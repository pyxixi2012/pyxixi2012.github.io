JavaScript Development Environment

在ubuntu16下安装 javasrcipt 开发环境 ，包括Node.js ,angular.js,npm ,gulp 等

引用资源

Installing git

$ sudo apt-get install github: “https://github.com/pyxixi2012“

$ sudo apt install aptitude

Installing nodejs

Install nodejs with nvm

$nvm –version

$nvm install v6.3.1

$node -v

v6.3.1

Install nodejs from tar files

$wget https://nodejs.org/dist/v6.3.1/node-v6.3.1-linux-x64.tar.xz

$ tar zvxf node-v6.3.1-linux-x64.tar.xz

$ cd node-v6.3.1-linux-x64

$ sudo mv node-v6.3.1-linux-x64 /usr/local/node

$ sudo ln -s /usr/local/node/bin/node /usr/bin/node

$ sudo ln -s /usr/local/node/bin/npm /usr/bin/npm

$ node -v

v6.3.1

Setting Up Express

$ npm install -g express

$ sudo apt install node-express-generator

eixit current terminal and reopen a new terminal

$ express myproject

$ cd myproject

$ npm install

$ npm start

curl -i http://127.0.0.1:3000
